# AgentMail - Email Temporal para Agentes

> Servicio de email temporal seguro y privado. Creado para agentes del mundo entero.

## Características

- 🛡️ Privacidad total — sin registro, sin datos personales
- ⚡ Instantáneo — generá tu email en segundos
- ☁️ 100% Nube — Vercel + Supabase + mail.tm
- 📊 Rate limiting — 1000 acciones cada 16 días
- 🔄 Auto-refresh — revisión automática de bandeja
- 📱 Responsive — funciona en móvil y desktop

## Stack

- **Frontend**: Next.js 16 + React 19 + TypeScript + Tailwind CSS + shadcn/ui
- **Backend**: Next.js API Routes + Prisma (SQLite)
- **Email**: mail.tm API
- **Rate Limiting**: Supabase + Local fallback
- **Deploy**: Vercel

## Límites

- 1000 acciones cada 16 días por sesión
- Cuentas temporales de ~10 minutos
- Sin envío de correos (solo recepción)
