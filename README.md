# AgentMail - Envío de Email para Agentes

> Servicio de envío de email profesional para agentes del mundo. Enviá emails con formato HTML a través de Resend.

## Características

- 📤 Envío profesional — emails HTML con Resend
- ⚡ Instantáneo — componé y enviá en segundos
- 🎨 Plantillas — templates pre-diseñados para uso común
- 👁️ Vista previa — revisá antes de enviar
- ☁️ 100% Nube — Vercel + Supabase + Resend
- 📊 Rate limiting — 1000 envíos cada 16 días
- 📱 Responsive — funciona en móvil y desktop

## Stack

- **Frontend**: Next.js 16 + React 19 + TypeScript + Tailwind CSS + shadcn/ui
- **Backend**: Next.js API Routes
- **Email Sending**: Resend API
- **Rate Limiting**: Supabase + Local fallback
- **Deploy**: Vercel

## Configuración

1. Obtené una API key gratuita en [resend.com/signup](https://resend.com/signup)
2. Configurala en Ajustes dentro de la app
3. ¡Empezá a enviar emails!

## Límites

- 1000 envíos cada 16 días por sesión
- Plan gratuito de Resend: 100 emails/día, 3,000 emails/mes
- Envío desde `onboarding@resend.dev` (dominio de prueba)

Ver la sección "Reglas" en la aplicación para las condiciones completas.
